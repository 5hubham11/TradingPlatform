# TradingPlatform - Zerodha Clone

TradingPlatform is a full-stack stock trading application inspired by Zerodha, designed to simulate a modern online brokerage platform. The project consists of a public-facing landing website and a protected trading dashboard integrated through secure JWT-based authentication.

## Features

### User Authentication

* User Registration and Login
* JWT Authentication using HTTP-only Cookies
* Protected Routes
* Session Persistence
* Secure Logout Functionality

### Landing Platform

* Responsive Zerodha-inspired UI
* Home, About, Products, Pricing, and Support Pages
* User Profile Navigation
* Dynamic Authentication-Based Navbar

### Trading Dashboard

* Portfolio Overview
* Holdings Management
* Positions Tracking
* Orders Dashboard
* Funds Management
* Watchlist Interface
* Buy Order Popup Window
* User-Specific Dashboard Access

### Backend Services

* RESTful API Architecture
* Express.js Server
* MongoDB Database Integration
* Mongoose ODM
* Authentication Middleware
* Secure Token Generation and Verification
